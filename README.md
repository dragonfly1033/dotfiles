# dotfiles

1. Perform a base install up until the arch-chroot
2. Install git
3. Clone this Repo
4. ``` chmod +x .dotfiles/install/install_* ```
5. Run install_1.sh
6. Follow instructions provided at the end.
7. Log in as your user
8. Run install_2_sudo.sh with sudo
9. Run install_3.sh
10. Reboot
